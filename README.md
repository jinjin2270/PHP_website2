Future Blog - Complete PHP Blogging Platform

A fully-featured, premium blogging platform built with PHP, MySQLand modern web technologies.
This platform enables users to create, share and discover content with role-based access control, premium subscriptions with an elegant user interface.

Features
For All Users

Responsive Design - Fully responsive interface that works on all devices
Category Navigation - Browse posts by categories
Popular Posts - View most liked and commented posts
Search Functionality - Find content by keywords
User Authentication - Secure login and registration system
Social Sharing - Share posts on social media platforms

For Registered Users

Create Posts - Submit new blog posts with images
Like Posts - Interact with content through likes
Comment System - Engage in discussions through comments
Edit Profile - Update personal information and password
Premium Content Access - View exclusive premium posts (with subscription)

For Premium Members

Exclusive Content - Access to premium-only posts

For Administrators

Admin Dashboard - Centralized control panel
User Management - Create, edit, and delete users
Post Review - Approve or reject submitted posts
Category Management - Add, edit, and delete categories
Comment Moderation - Remove inappropriate comments
Role Management - Assign user roles and permissions

Technology Stack

Backend: PHP 7.4+
Database: MySQL 5.7+
Frontend: HTML5, CSS3, JavaScript (ES6+)
Styling: Custom CSS with CSS Variables
Icons: Font Awesome 6
Fonts: Google Fonts (Manrope, Playfair Display)
Security: Password hashing, Session management, Input validation

Prerequisites
PHP 7.4 or higher

MySQL 5.7 or higher

Apache/Nginx web server

mod_rewrite enabled (for Apache)

PHP extensions: mysqli, session, fileinfo
